# GestionDepenses – Bonjour futur millionnaire !

Application Android de gestion de budget intelligente développée en Kotlin.

## Fonctionnalités
- Saisie rapide des dépenses
- Alertes rouges/oranges quand tu dépasses ton salaire ou +30 % vs mois dernier
- Graphiques magnifiques (Bar + Pie Chart)
- Message motivant à chaque ouverture : **"Bonjour futur millionnaire"**
- 100 % offline (base Room)
